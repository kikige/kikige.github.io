购买域名

推荐使用GoDaddy，支持支付宝，银联付款，操作简单。

  1. 注册GoDaddy账号
  2. 查找想要的域名。未被注册的即可选择加入购物车。
    注：GoDaddy附加收费服务，根据需要勾选。不熟悉英文的同学，左上方国家变为新加坡，相应的语言变为简体中文。注意有时选择不同的国家优惠价格不同。
  3. 确认购买。初次购买可以通过搜索引擎，搜索一些新鲜日期的优惠码使用。
  4. 结算。填写完必要信息后，可通过银联、支付宝或信用卡结算。
  5. 检查。重新登陆GoDaddy账户，查看购买的域名是否显示在你的账户下。
  6. 激活。在注册所用邮箱中，Godaddy激活邮件，确认激活后方可使用。
